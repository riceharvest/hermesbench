# Anchor tasks

Anchor tasks are stable longitudinal HermesBench tasks. A task may enter this directory only after at least one public/dev or fresh run window, deterministic/test scoring review, and confirmation that it does not depend on current-world facts.

Changes to anchor prompts, fixtures, or scoring require a new benchmark version. Deprecations must be recorded in the changelog and in release notes. See `docs/anchor-set-policy.md`.
