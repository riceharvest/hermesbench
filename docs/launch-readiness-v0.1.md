# HermesBench v0.1 launch readiness

HermesBench must not claim official leaderboard integrity until every required gate is satisfied.

- [ ] At least 30 public/dev tasks are real, fixture-backed, deterministic or hybrid.
- [ ] At least 20 private holdout tasks exist outside public repo.
- [ ] At least 10 fresh tasks exist for current wave.
- [ ] At least 10 anchor tasks are frozen and versioned.
- [ ] API accepts uploads and stores submissions without tokens.
- [ ] Website has leaderboard and result detail pages.
- [ ] Official run policy is published.
- [ ] Result archive command works.
- [ ] CI is green.
- [ ] Public task export release artifact exists.
- [ ] Private pack handling tested locally by maintainer.
- [ ] First official run archived with manifest and hash.

## Go/no-go rule
If any box is unchecked, publish only as scaffold, prerelease, or unofficial public/dev demo.
